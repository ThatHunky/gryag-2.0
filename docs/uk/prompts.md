# Промпти та Особистість

Бот Гряг має унікальну особистість, яка визначається системними промптами. Це не просто асистент, а повноцінний учасник чату зі своїм характером.

## Особистість Гряг

- **Ідентичність**: Гряг (@gryag_bot) — український хлопець у груповому чаті.
- **Стиль спілкування**: Розмовна українська мова, неформальний стиль, використання сленгу та ідіом.
- **Характер**: Гострий на язик, саркастичний, з чорним гумором. Він має власну думку і не боїться її висловлювати.
- **Принципи**: Підтримує Україну, уникає офіціозу, спілкується як реальна людина.

## Доступні промпти

У каталозі `prompts/` містяться різні варіанти системних налаштувань:

1. **`default.md`**: Основний промпт. Визначає базову поведінку, стиль спілкування та правила використання інструментів.
2. **`assistant.md`**: Більш стриманий та помічний варіант для специфічних завдань.
3. **`casual.md`**: Максимально розслаблений стиль для дружнього спілкування.

## Динамічний Контекст

Системний промпт автоматично доповнюється динамічними даними:

- **Інформація про чат**: Назва, тип, кількість учасників.
- **Дані користувача**: Ім'я, юзернейм, звертання.
- **Пам'ять**: Факти про користувача, які бот запам'ятав у минулому.
- **Сумаризація**: Стислий зміст попередніх розмов для підтримки довготривалого контексту.

## Налаштування

Ви можете змінити активний файл промпту в `.env` (змінна `SYSTEM_PROMPT_FILE`). Файли промптів монтуються в контейнер як `read-only` для стабільності.
